Endpoints

Here is a list of available endpoints:

  • POST /api/stocks

POST /api/stocks

Get all stock data (including dividends etc.) from start date to end date.

Request

POST /api/stocks
{
    "stock_code": "MAVI"
    "start_date": "1514764800" //  default value 01/01/2018
    "end_date": "TODAY's Timestamp" // default value
}
