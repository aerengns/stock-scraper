Stock Scraper API

Introduction

Welcome to the API documentation. Here you will find all the information you need to interact with our API.

Authentication

There is no need of authentication at the moment.

Endpoints

Here is a list of available endpoints:

  • POST /api/stocks
  • POST /api/currency

POST /api/stocks

Get all stock data (including dividends etc.) from start date to end date.

Request

POST /api/stocks
{
    "stock_code": "MAVI"
    "start_date": "1514764800" //  default value 01/01/2018
    "end_date": "TODAY's Timestamp" // default value
}

{

    "history_data": [

POST /api/currency

Convert currency from one to another at a specific date.

Request

POST /api/currency
{
    "from_currency": "TRY"
    "to_currency": "USD"
    "amount": 100
    "date": "2023-05-12T13:22:58.898Z" // default value TODAY
}

Response

{
    "converted_amount": 5.10872
    "conversion_rate": 0.0510872
}
